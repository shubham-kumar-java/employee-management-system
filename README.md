# Employee Management System

This is a backend project built using Spring Boot and MySQL.

## Technologies Used
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Create employee
- Get all employees
- Get employee by ID
- Delete employee

## How to Run
1. Create database `ems` in MySQL
2. Update username & password in application.properties
3. Run the main class
4. Use Postman to test APIs

## APIs

POST /employees  
GET /employees  
GET /employees/{id}  
DELETE /employees/{id}
